VAMPS Required Metadata Terms (follows the Earth Microbiome Project metadata guide)


  • Taxon_id (metagenome)
          see NCBI Taxonomy Browser
  • Description (of sample)
  • Common_name
  • Altitude
  • Assigned_from_geo ('y' or 'n')
         -from geographic source?
  • Collection_date
         Format: YYYY-MM-DD
  • depth (meters below surface)
  • elevation (meters above sea level)
  • latitude (decimal degrees)
  • longitude (decimal degrees)
