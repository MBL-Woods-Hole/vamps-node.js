1) Select the datasets and/or customized dataset combinations you would like to analyze from the list on the left.
2) Choose Simple or Custom taxonomy, and select the domains and taxonomic depth for your analysis.
3) Choose normalization (to maximum or frequency) for the data comparison. Default is unnormalized.
4) Select the analysis from the series below.

  • Heatmap
  • Counts (taxonomy) Table
  • Bar Charts
  • Dendrograms
  • Alpha Diversity
